## 性能优化

默认的设置对电脑性能要求较高，如果您的电脑性能较低，您可以尝试以下方法来优化性能：

- 移除草地、树叶材质包和SPBR材质包。
- 关闭着色器。
- 禁用Distant Horizons模组。
